# mainPage
本專案為 AIStreamer 專題中的直播主視覺顯示介面（MainPage），設計目標是打造一個簡潔直觀的直播畫面，整合了視訊串流區、即時聊天室、AI 觀眾反應模擬，以及視覺化數據（例如即時觀看人數變化圖表）等模組，提供直播者互動與內容回饋依據。
Requirements:
Node.js & npm (we recommend installing via nvm)

Steps:
bash
複製
編輯
